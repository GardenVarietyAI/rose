# Build Scripts

## build.sh

- Builds both JavaScript and CSS bundles using esbuild via Docker Compose.

**Usage:**
```bash
mise esbuild
# or
./scripts/build.sh
```

## ajv.sh

- Generates JSON Schema + standalone Ajv validators used by the frontend.

**Usage:**
```bash
mise ajv
# or
./scripts/ajv.sh
```
